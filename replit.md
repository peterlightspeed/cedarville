# Cedarville Private Schools Website

## Overview
Fully static school website for Cedarville Private Schools in Lagos, Nigeria (founded 2014). Built with Bootstrap 5, vanilla JavaScript, and a rule-based chatbot. No database required.

## Pages
- index.html - Homepage
- about.html - About the school
- admissions.html - Enrollment info and fees
- academics.html - Programs (Creche, Nursery, Kindergarten)
- gallery.html - Photo gallery with lightbox and category filter
- blog.html - Blog posts with category filter
- contact.html - Contact form and info

## Technology Stack
- **Frontend**: HTML5, Bootstrap 5.3.3, CSS3, Vanilla JavaScript
- **Backend**: Node.js + Express.js (serves static files only)
- **Chatbot**: Rule-based keyword matching (no API, works offline)
- **CDN**: Bootstrap, Font Awesome 6.5.1

## Key Files
- css/style.css - All styles (blue and white theme with CSS variables)
- js/chatbot.js - Rule-based chatbot with keyword responses
- js/main.js - Global JS: counters, gallery filter, blog filter, contact form, lightbox
- TODO.md - Full guide for updating content, images, logo, and links

## Running the Site
```bash
npm start
```
Site runs on port 5000.
